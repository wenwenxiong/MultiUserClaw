# Postgres表结构
⏺ 当前 users 表结构：
  ┌───────────────┬─────────────────────┬────────────────────────────────────────┐
  │     字段      │        类型         │                  说明                  │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ id            │ varchar(36) PK      │ UUID 主键                              │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ username      │ varchar(64) UNIQUE  │ 用户名（SSO 用户取自 trueName）        │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ email         │ varchar(256) UNIQUE │ 邮箱（SSO 用户为 {uid}@infox-med.sso） │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ password_hash │ varchar(256)        │ bcrypt 密码（SSO 用户为随机值）        │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ role          │ varchar(16)         │ user / admin                           │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ quota_tier    │ varchar(16)         │ free / basic / pro                     │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ is_active     │ boolean             │ 是否启用                               │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ created_at    │ timestamp           │ 创建时间                               │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ updated_at    │ timestamp           │ 更新时间                               │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ sso_uid       │ varchar(64) UNIQUE  │ SSO 用户 ID（如 1106970）              │
  ├───────────────┼─────────────────────┼────────────────────────────────────────┤
  │ sso_token     │ varchar(256)        │ SSO token（用于注入容器）              │
  └───────────────┴─────────────────────┴────────────────────────────────────────┘
